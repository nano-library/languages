La famille (Family)

Les parents (Parents)

  • le père (father)
  • la mère (mother)

Le couple (Couple)

  • le mari (husband)
  • la femme (wife)

Les enfants (Children)

  • le fils (son)
  • la fille (daughter)

Les frères et sœurs (Brothers and sisters / Siblings)

  • le frère (brother)
  • la sœur (sister)

Les grands-parents (Grandparents)

  • le grand-père (grandfather)
  • la grand-mère (grandmother)

Les petits-enfants (Grandchildren)

  • le petit-fils (grandson)
  • la petite-fille (granddaughter)

Les beaux-parents (Parents-in-law)

  • le beau-père (father-in-law / stepfather)
  • la belle-mère (mother-in-law / stepmother)

Les cousins (Cousins)

  • le cousin (male cousin)
  • la cousine (female cousin)

Les arrière-grands-parents (Great-grandparents)

  • l'arrière-grand-père (great-grandfather)
  • l'arrière-grand-mère (great-grandmother)

Les arrière-petits-enfants (Great-grandchildren)

  • l'arrière-petit-fils (great-grandson)
  • l'arrière-petite-fille (great-granddaughter)

Les beaux-frères (In-laws: brothers/sisters-in-law)

  • le beau-frère (brother-in-law) (“beauf” in informal French)
  • la belle-sœur (sister-in-law)

Les oncles et neveux (Uncles and nephews/nieces)

  • l'oncle (uncle)
  • la tante (aunt)
  • le neveu (nephew)
  • la nièce (niece)
